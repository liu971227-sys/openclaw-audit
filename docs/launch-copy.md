## Hacker News / Reddit Style

I built `openclaw-audit`, a local security audit and hardening CLI for OpenClaw.

It checks:
- gateway exposure
- auth and Control UI weaknesses
- trusted-proxy mistakes
- browser SSRF / remote CDP risk
- synced-folder and symlink state-dir problems
- secret leakage in config, sessions, and logs

It also generates a Caddyfile, a hardening guide, and a config fix-preview draft.

Repo: https://github.com/liu971227-sys/openclaw-audit
